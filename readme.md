# mp4-recorder


### 简介

> using lib-mp4v2 and lib-fdk-aac to record mp4 file


### 基本设计概念以及处理流程

> pcm + h264 = mp4


### 模块
- libmp4v2
    > https://code.google.com/archive/p/mp4v2/
- libfdk-aac
    > https://sourceforge.net/projects/opencore-amr/files/fdk-aac/
- Mp4Recorder
    > class of MP4 recorder
